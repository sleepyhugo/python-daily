# Password Generator

A simple command-line password generator built with Python.

## Features
- Generate random passwords with letters, numbers, and symbols
- Choose custom password length
- Generate multiple passwords in one session

## How to Run
1. Run: `python password_generator.py`
2. Enter desired password length
3. Choose whether to generate another
